CohortConnect

Project Overview

CohortConnect is an advanced desktop address book developed by a team as part of CS2103T. It's designed for Computer Science students, offering a Command Line Interface (CLI) with the benefits of a Graphical User Interface (GUI). The address book can manage large contact groups, features easy import, the ability to mark favorite contacts, and helps users find contacts attending the same events. Additionally, it facilitates networking with like-minded students through a "Find A Buddy" feature that matches users based on similar interests using GitHub's metadata and a proprietary algorithm.
